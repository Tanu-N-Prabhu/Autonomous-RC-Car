<h2> The materials required for this project are as follows.</h2>

1. RC Car
2. Ultrasonic Sensor.
3. Ada-fruit Motor Shield L239D.
4. Servo.
5. LCD. 
6. LED for the brake lights.
7. Massive amount of batteries.
